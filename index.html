Operaciones Varias

Calculadora

Descripción:

Calculadora básica de operaciones matemáticas declarando dos números y seleccionando la operación requerida.

Instrucciones:

Escribir los datos en recuadro, se le solicitara un numero a operar, luego de aceptar deberá ingresar un operador. Eligiendo entre +(Suma), -(Resta), *(Multiplicación) /(Division). Para realizar la operación al finalizar se le solicitara un Segundo numero para completar la operación.

Calculador de Promedio de Goles por partido.

Descripción:

Calculador para Obtener un promedio de goles por partido de un Jugador/a

Instrucciones:

Escribir los datos en recuadro, se le solicitara el nombre del jugador/a a promediar, después se le solicitara la cantidad de partidos jugados y para finalizar el calculo se le solicitara la cantidad de goles del jugador.

Calculador de Promedios de notas.

Descripción:

Calculador para Obtener un promedio de notas en tres parciales, obteniendo la calificación final y tipo de final a rendir.

Instrucciones:

Escribir los datos en recuadro, se le solicitara las notas de los 3 parciales, luego de aceptar se le devolverá el promedio y tipo de Final a rendir en base a sus calificaciones.
